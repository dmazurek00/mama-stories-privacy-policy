# Privacy Policy – Mommy Stories

Last updated: 2026-04-02

## 1. Overview

Mommy Stories is an Android application that allows users to record spoken stories or voice notes, upload them for processing, and receive structured text output based on those recordings.

This Privacy Policy explains how Dawid Mazurek (“we”, “us”, “our”) accesses, collects, uses, stores, and shares data when you use the app.

## 2. Developer and Contact

## 3. Data We Access and Process

Depending on how you use the app, the app may access and process:

- microphone input and audio recordings created by the user
- text generated from those recordings, including transcription and edited story text
- technical metadata required for app functionality, such as app version, device/platform information, timestamps, and upload-related technical data
- email delivery data if the app sends finished content by email

## 4. Why We Process This Data

We process this data only to provide the core functionality of the app, including:

- recording audio
- uploading audio securely to the backend
- transcribing and processing audio into text
- improving formatting and structure of the resulting story or text
- delivering the result to the user
- maintaining reliability, security, and basic service operations

## 5. Microphone Permission

The app requests `RECORD_AUDIO` permission only so the user can intentionally record audio inside the app.

The microphone is not used secretly or continuously in the background. It is used only when the user starts a recording action.

## 6. Data Sharing and Third Parties

To provide the app’s functionality, data may be processed by third-party service providers acting on our behalf, such as:

- hosting / server infrastructure providers
- AI processing providers used for transcription and text processing
- email delivery providers, if email sending is part of the app’s functionality

We do not sell personal data.

## 7. Data Storage and Retention

Audio recordings, generated text, and related technical metadata may be stored for as long as necessary to provide the service, maintain archives, troubleshoot issues, and support the user experience, unless deletion is requested and can be fulfilled, or a shorter retention period is required by law.

## 8. Security

We use reasonable technical and organizational measures to protect data, including secure transport and controlled access where applicable.

However, no method of transmission or storage is completely secure, and absolute security cannot be guaranteed.
